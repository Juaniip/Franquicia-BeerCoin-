# Franquicia-BeerCoin-

|| BeerCoin – Sitio Web Oficial||

- Descripción:

Este repositorio contiene el código fuente del sitio web de BeerCoin, una franquicia de bares innovadora cuyo concepto gira en torno a un sistema de expendio de cerveza por monedas, combinando tecnología, autoservicio y experiencia social.
El sitio está diseñado para comunicar la identidad de la marca, mostrar el funcionamiento del sistema BeerCoin y presentar los espacios del local, como áreas exteriores, salas multiuso y cabinas para streaming o música en vivo.

- Concepto BeerCoin:

BeerCoin propone una experiencia diferente:
el bar funciona con un sistema de monedas físicas o tokens que activan el expendio de cerveza mediante un botón, eliminando filas y agilizando el consumo, manteniendo siempre el foco en la experiencia social.
